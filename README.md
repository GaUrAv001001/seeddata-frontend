# Frontend Part of Assement
This is the frontend application for the assessment project developed for Roxiler System. The application uses React and Vite for a fast and modern development experience.

Project is Hosted at: https://seeddata-frontend.onrender.com

## Installation

1. git clone <repository-url>
2. cd project-folder

## Scripts

1. Run the development server: `npm run dev`
2. Build the application for production: `npm run build`
3. Preview the production build: `npm start`

## Components

The following components are included in this project:

- `BarChartStats:` Displays bar chart statistics.
- `Home:` The main landing page of the application.
- `PieChartStats:` Displays pie chart statistics.
- `Statistics:` Aggregates and presents various statistical data.
- `Footer:` The footer section of the application.
- `Header`: The header section of the application, including navigation.

## Dependencies

This project uses the following dependencies:

- `axios:` For making HTTP requests.
- `chart.js:` A flexible JavaScript charting library.
- `react:` A JavaScript library for building user interfaces.
- `react-chartjs-2:` A wrapper for Chart.js in React.
- `react-dom:` Provides DOM-specific methods that can be used at the top level of your app.
- `react-icons:` Includes popular icons as React components.
- `react-router-dom:` Enables routing in the application.

## Usage
After installing the dependencies and starting the development server, you can access the application at http://localhost:5173. The hosted version of the project can be accessed at:
